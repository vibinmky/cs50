Git commands

Command Action
git clone <url> clone the remote repository to local machine
git add <File Name> Track the added file changes
git commit -m "messages" Do a commit with the messages
git status Basic Details about our working branch
git push Push your changes to the repository
git pull copy the remote repository to the local machine
git log Show the log of commits happend
git reset --hard <commit code> go back to previous mentioned state of code
git reflog All the git event records
git branch Show the branch details and current working branch
git branch <new branch name> create a new branch
git checkout <branch name> move to the specified branch
git merge <branch name> merge branch with the master branch
git push --set-upstream origin <branch name>
